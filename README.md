# CareerGPT

CareerGPT is an AI-powered Career Guidance Platform that analyzes resumes and provides personalized career recommendations.

## Features

- Resume Parsing from PDF
- AI Skill Extraction
- Role Compatibility Analysis
- Skill Gap Detection
- AI Resume Advisor
- Personalized Career Roadmap
- Interview Question Generator
- Learning Resource Recommendation

## Tech Stack

- Python
- Streamlit
- Groq API
- Pandas
- PyPDF


## Supported Roles

- AI Engineer
- Machine Learning Engineer
- Data Scientist
- Frontend Developer
- Backend Developer
- Cybersecurity Analyst

pip install -r requirements.txt

streamlit run app.py

Free-tier Groq accounts may occasionally hit token or rate limits during heavy usage. In such cases, please wait a few minutes and try again.
